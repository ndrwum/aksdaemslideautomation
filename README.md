# aksdaemslideautomation
A Google Slides script that automates the creation of church presentation slides. This tool:

Pulls data from a Google Sheets document containing assigned duties for each Sabbath day.
Retrieves the scripture reading reference, scrapes the corresponding verse text from an online source, and inserts it into the appropriate slide.
Fetches hymnal numbers, scrapes the lyrics from an online source, and adds them to the respective slide.
